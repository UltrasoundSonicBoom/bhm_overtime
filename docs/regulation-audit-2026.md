### BUG-02: 장기근속수당 25년+ 금액 오류

| 항목 | 내용 |
|------|------|
| **파일:위치** | `data.js` L181 |
| **현재 코드** | `{ min: 25, max: 99, amount: 130000 }` |
| **규정 근거** | 제50조 장기근속수당: "20년 이상 10만원. 21년 이상 1만 가산, 25년 이상 3만 가산" |
| **올바른 계산** | 100,000(20년+) + 10,000(21년 가산) + 30,000(25년 가산) = **140,000원** |
| **수정 방법** | `amount: 130000` → `amount: 140000` |
| **영향** | 25년 이상 재직자 월 10,000원 과소 계산 |

### BUG-03: 장기근속수당 구조 불일치 (TIER vs ADDITIVE)

| 항목 | 내용 |
|------|------|
| **파일:위치** | `data.js` L173~182 |
| **현재 구조** | TIER: `{min:20, max:21, amount:100000}`, `{min:21, max:25, amount:110000}`, `{min:25, max:99, amount:130000}` |
| **규정 근거** | 제50조: "가산" 표현 → ADDITIVE (기존액에 더하는 방식) |
| **올바른 구조** | ADDITIVE: 20년 이상 +100,000 / 21년 이상 추가 +10,000 / 25년 이상 추가 +30,000 |
| **수정 방법** | TIER 값을 올바른 합계로 수정 (20년=100K, 21년=110K, 25년=140K) |

> BUG-02와 BUG-03은 연결. BUG-03 구조 확인 후 BUG-02 값 수정.
