### 퇴직수당 (제53조) — 2015.6.30 이전 입사자 한정
월평균임금 × 근속연수 × 비율

| 근속연수 | 비율 |
|---------|-----|
| 1년~5년 미만 | 10% |
| 5년~10년 미만 | 35% |
| 10년~15년 미만 | 45% |
| 15년~20년 미만 | 50% |
| 20년 이상 | 60% |

### 임금피크제 (만60세 적용, 2024년~)
기준: 만60세 직전 3개월 평균임금

| 옵션 | 내용 |
|-----|-----|
| A (공로연수) | 1년간 공로연수 + 60% 지급 (근무 없음) |
| B (계속근무) | 계속 근무 + 100% 지급 |

→ 옵션 A가 퇴직금 산정에 유리한지 여부 계산 포함

## 탭 구성 (retirement.html)

### 탭 1: 내 퇴직금
- 입력: 월평균임금, 입사일(또는 근속연수), 퇴직일
- 계산: 퇴직금 + 퇴직수당(2015.6.30 이전 입사자 체크박스)
- 2001.8.31 이전 근속분 있으면 인사과 문의 배너 표시

### 탭 2: 임금피크 비교
- 입력: 만60세 직전 월평균임금
- 옵션 A vs B 비교 카드 (수령액, 퇴직금 산정 차이)

### 탭 3: 사학연금
- 사학연금 개요 (국민연금→사학연금 전환 배경)
- 수령 나이: 65세 (조기수령 가능)
- 외부 시뮬레이션 링크: https://www.tp.or.kr (사학연금공단)
- 중간정산 관련: 인사과 문의 안내

### 탭 4: 퇴직 준비 체크리스트
- 퇴직 D-90 / D-30 / D-7 체크리스트
- 인사과 제출서류, 퇴직정산 절차 등 텍스트 안내

## 디자인 시스템
- `<html data-theme="neo">` — neo-brutalist 테마
- `<link rel="stylesheet" href="style.css">` (v2.5)
- `shared-layout.js` 포함 (헤더 + 바텀 nav)
- 하드 보더: `border: 3px solid #1a1a1a`, `box-shadow: 6px 6px 0px #1a1a1a`
- 탭 UI: `.pay-bookmark-tabs` / `.pay-bookmark-tab` 클래스 재사용 (또는 동일 패턴)
- 카드: `.card` 클래스
- 버튼: `.btn.btn-primary`
- 색상 토큰: `--accent-indigo`, `--accent-emerald`, `--text-primary`, `--text-muted`

## 보안 주의사항
- XSS hook이 `innerHTML`에 동적 문자열 주입 차단
- 모든 동적 렌더링은 `element.textContent`, `createElement`, `appendChild` 사용
- `retirement-preview.html`이 DOM-safe 참조 구현체
