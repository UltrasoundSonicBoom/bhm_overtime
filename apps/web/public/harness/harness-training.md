# 교육 콘텐츠 자동 생성 팀 하네스

직원 교육 목표를 입력하면 커리큘럼·학습 자료·평가 문항을 에이전트팀이 완성한다.

## 에이전트 구성

```
🧠 오케스트레이터
    │
    ├── 📐 커리큘럼설계 에이전트
    │     교육 목표 → 학습 목표 분해 (블룸 분류)
    │     선수학습·순서·시간 배분 계획
    │
    ├── ✍️ 콘텐츠생성 에이전트
    │     각 학습 목표 → 교육 자료 생성
    │     텍스트·사례·시뮬레이션·퀴즈 형식 혼용
    │
    └── 🔍 QA검증 에이전트
          사실 정확성 검증 (의학 지식 DB 대조)
          규정 부합성 확인 (단협·법령)
          학습 목표 커버리지 평가
```

## 실행 흐름

```
/create-training --target="신규 간호사" --topic="낙상 예방" --duration="4시간"

Step 1: 커리큘럼설계 → 4시간 학습 계획 (30분 단위)
Step 2: 콘텐츠생성 → 각 모듈별 교재 초안
Step 3: QA검증 → 사실 확인 + 법적 적합성
Step 4: 오케스트레이터 → 통합 패키지 완성
Step 5: 담당자 최종 검토 후 배포

제작 시간: 1주 → 하루 (자동화 부분 기준)
```

## 출력 형식

- PDF 교재 (인쇄용)
- 슬라이드 덱 (발표용)
- 평가 문항 20-30개 (지식 확인)
- e-Learning 모듈 (LMS 연동용 SCORM)

## 적용 직종

- **관리자·파트장** (교육 프로그램 기획)
- **간호사** (신규 교육 콘텐츠 수혜)
