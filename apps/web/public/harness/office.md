# 업무 생산성 (Office Productivity) — 병원 AI 패턴

병원 행정·운영 업무의 반복 작업을 자동화하고 정보 접근성을 높이는 AI 패턴 모음.

## 포함 패턴

| 패턴 | 핵심 기능 | SNUHmate 연동 |
|-----|---------|-------------|
| 회의록 STT | 음성→구조화 요약·액션아이템 | — |
| 규정 검색 | 단협·법령 조항 즉시 검색 | 찾아보기 탭 |
| 인수인계 자동화 | 교대 메모→표준화 인수인계서 | — |
| 급여명세서 분석 | 항목 한글 해설·오류 감지 | 급여 탭 B4 |
| PII 마스킹 | Presidio→LLM 2단계 자동 마스킹 | B6 마스킹 |
| 커리어 트윈 | 유사 경력 동료 진로 경로 매핑 | B2 커리어 트윈 |
| 부서 위키 | 업무 절차 자동 문서화·AI 검색 | B9 부서 위키 |

## 적용 직종

- **간호사** (인수인계, 규정 검색)
- **행정사무직** (회의록, 부서 위키, PII 마스킹)
- **관리자·파트장** (보고서, 커리어 관리)
- **의료기사·보건직** (급여 분석, 커리어 트윈)

## Claude Code 구현 가이드

```
규정 검색 RAG 패턴 (SNUHmate 찾아보기 탭 방식):
1. 단협 문서를 청크 단위로 분할·인덱싱
2. 사용자 질의 → 의미 기반 검색 (동의어 매핑 포함)
3. 관련 조항 추출 + 조항 번호 인라인 표시
4. 단계적 세부 내용 확장 (바텀시트 패턴)

급여명세서 분석 패턴:
1. 명세서 이미지·PDF → OCR 파싱
2. 각 항목 약어 → 한글 명칭 매핑 (단협 기준)
3. 계산 공식 검증 (단협 조항 참조)
4. 전월 대비 차이 하이라이트
```

## SNUHmate 현재 구현 상태

- ✅ **규정 검색** — 찾아보기 탭 (동의어·공백무시 검색 완료)
- ✅ **급여 시뮬레이터** — 급여 탭 (호봉표 기반 계산)
- 🔧 **PII 마스킹 (B6)** — 설계 완료, 구현 대기
- 🔲 **커리어 트윈 (B2)** — 페르소나 인터뷰 완료, 구현 예정
- 🔲 **명세서 AI 분석 (B4)** — 페르소나 완료, 구현 예정
- 🔲 **부서 위키 (B9)** — Schedule Tab Phase 3 연동 예정
