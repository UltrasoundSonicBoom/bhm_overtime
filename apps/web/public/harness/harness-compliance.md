# 규정 준수 모니터링 팀 하네스

법령·단협·병원 정책 변경을 실시간으로 추적하고 영향받는 부서에 맞춤 대응 방안을 자동 배포한다.

## 에이전트 구성

```
🧠 오케스트레이터
    │
    ├── 📡 규정추적 에이전트
    │     법령 정보 시스템·단협 DB 모니터링
    │     변경 감지 시 오케스트레이터 트리거
    │
    ├── 🔍 영향분석 에이전트
    │     변경 내용 → 영향받는 부서·직종 매핑
    │     기존 정책과 충돌 여부 확인
    │
    └── 📣 알림발송 에이전트
          부서별 맞춤 요약 작성
          긴급도 기반 알림 채널 선택 (SMS·앱·이메일)
```

## 실행 흐름

```
자동 트리거: 모니터링 주기 (1일 1회)
또는 /compliance-check [특정 법령명]

Step 1: 규정추적 에이전트 → 변경 감지
Step 2: 영향분석 에이전트 → 대상 부서 목록
Step 3: 알림발송 에이전트 → 맞춤 공지 발송
Step 4: 오케스트레이터 → 조치 완료 확인

SNUHmate 단협 검색 탭 자동 업데이트 연동 가능
```

## 추적 대상

- 의료법·간호법·의료기사법 개정
- 산업안전보건법 병원 적용 사항
- SNUH 단체협약 변경
- 개인정보보호법 의료 관련 고시

## 적용 직종

- **관리자·파트장** (규정 준수 책임자)
- **행정사무직** (인사·총무)

## SNUHmate 연동

- 찾아보기 탭 규정 DB와 동기화
- 단협 변경 시 계산 로직 업데이트 알림 → `/smate-payroll-review` 트리거
