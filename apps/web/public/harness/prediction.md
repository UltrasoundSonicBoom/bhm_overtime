# 예측·패턴 (Prediction & Patterns) — 병원 AI 패턴

EMR 데이터·근무 기록을 분석해 위험을 사전에 감지하고 예방 조치를 제안하는 패턴.

## 핵심 예측 모델

| 예측 대상 | 주요 입력 변수 | 임계값 액션 |
|---------|------------|----------|
| 낙상 위험 | 연령·약물·이동능력·입원일수 | 고위험 → 낙상예방 세트 자동 활성화 |
| 욕창 발생 | Braden 점수·체위변경·영양 | 위험 상승 → 간호사 즉시 알림 |
| 30일 재입원 | 진단·처방·사회적 지지 | 위험 60%↑ → 케어매니저 배정 |
| 직원 번아웃 | 월 초과근무·휴가소진·야간빈도 | 3개월 연속 초과 → 관리자 알림 |
| 약물 부작용 | 처방 조합·투여력·부작용 보고 | 반복 패턴 → 처방 경고 추가 |

## 적용 직종

- **간호사** (낙상·욕창 예방)
- **의사·전공의** (재입원·약물 안전)
- **관리자·파트장** (번아웃 감지, 인력 관리)
- **행정사무직** (번아웃 데이터 수집)

## Claude Code 구현 가이드

```
낙상 위험 예측 패턴:
1. EMR에서 위험 인자 데이터 수집 (배치 또는 실시간)
2. 규칙 기반 스코어링 (Morse Fall Scale 표준)
3. ML 모델 보완 (과거 낙상 사건 학습)
4. 임계값 초과 시 알림 생성
5. 예방 조치 체크리스트 자동 활성화
6. 이행 여부 추적

번아웃 감지 패턴:
1. 급여 시스템에서 초과근무 데이터 수집
2. 휴가 신청·승인 기록 분석
3. 야간·주말 근무 빈도 패턴 분석
4. 경보 기준 초과 시 관리자에게 대화형 알림
```

## 주요 기술 스택

- **스코어링 엔진**: 규칙 기반 (표준 사정 도구 기반)
- **ML 보완**: scikit-learn / LightGBM (과거 사건 학습)
- **실시간 알림**: 병원 알림 시스템 (SMS·앱 푸시)
- **번아웃 분석**: SNUHmate 초과근무 계산 모듈 연동

## SNUHmate 연동

- 초과근무 계산 탭 → 번아웃 감지 데이터 소스
- B4 (명세서 AI 분석) → 급여 이상 감지 연동 가능
