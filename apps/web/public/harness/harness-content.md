# 병원 콘텐츠 생성 팀 하네스

병원 뉴스·건강 정보·부서 소개 콘텐츠를 다양한 채널 형식에 맞게 에이전트팀이 자동 제작한다.

## 에이전트 구성

```
🧠 오케스트레이터
    │
    ├── 🔍 주제리서치 에이전트
    │     헬스케어 트렌드·계절성 질환·병원 이벤트 수집
    │     독자 관심도·공유 가능성 점수 산출
    │
    ├── ✍️ 초안생성 에이전트
    │     주제 → 의학적으로 정확한 초안 생성
    │     대상 독자(환자/직원/일반인) 맞춤 어조
    │
    └── 📱 채널최적화 에이전트
          SNS (인스타·카카오 2200자) 형식 변환
          웹 블로그 (SEO 최적화) 형식
          환자 뉴스레터 (카카오 알림톡) 형식
```

## 실행 흐름

```
/create-content --type="건강정보" --topic="가을 독감 예방" --channels="sns,web,newsletter"

Step 1: 주제리서치 → 최신 독감 현황 + 예방 가이드라인
Step 2: 초안생성 → 의학 정확도 검토된 본문
Step 3: 채널최적화 → 3가지 채널별 버전 동시 생성
Step 4: 담당자 검토 → 원클릭 배포

기존 대비: 1개 주제 → 3채널 동시 제작
```

## 콘텐츠 유형

- **건강 칼럼**: 월 4건 (환자 대상)
- **부서 소개**: 채용·대외 홍보
- **병원 뉴스**: 의료 성과·장비 도입 안내
- **의료진 소개**: 전문성 알리기

## 적용 직종

- **행정사무직** (홍보팀, 기획팀)
- **관리자·파트장** (부서 홍보)

## 의학적 정확성 보장

- 의학 정보는 최신 가이드라인 참조 (KCD·NECA·임상진료지침)
- 생성 콘텐츠에 "의료진 감수" 스탬프 자동 추가
- 민감한 의학 정보는 전문의 최종 확인 필수
