# 실시간 코파일럿 (Live Copilot) — 병원 AI 패턴

임상 현장에서 AI가 의료진과 함께 실시간으로 프로세스를 진행하는 패턴.

## 핵심 시나리오

| 코파일럿 유형 | 주요 기능 | 적용 환경 |
|------------|---------|---------|
| 핸즈프리 어시스턴트 | 음성 질의 → 약물용량·프로토콜 즉답 | 처치실·드레싱 중 |
| 수술실 체크리스트 | WHO SSC 진행·누락 알림·완료 기록 | 수술 시작 전 |
| 투약 안전 확인 | 바코드 스캔 + 환자ID 교차 확인 | 투약 직전 |
| 응급 트리아지 지원 | 주호소·바이탈 → KTAS 단계 즉시 제안 | 응급 내원 |

## 적용 직종

- **간호사** (처치실, 투약, 응급, 수술실)
- **의사·전공의** (수술, 응급 트리아지)
- **의료기사·보건직** (수술실 테크니션)

## Claude Code 구현 가이드

```
핸즈프리 코파일럿 패턴:
1. 음성 입력 수신 (Whisper STT)
2. 의도 분류 (용량계산 / 프로토콜조회 / 기록입력)
3. 의료 지식 베이스 조회 (정확도 우선)
4. 간결한 음성 응답 생성 (< 15초)
5. 대화 기록 EMR 연동 (선택적)

주의: 실시간 의사결정 지원 — 최종 판단은 의료진
```

## 주요 기술 스택

- **음성 인식**: Whisper (로컬), 응답 레이턴시 < 2초
- **의료 지식**: 약물 DB (DUR), 병원 프로토콜 문서 RAG
- **바코드 스캔**: 기기 카메라 + QR/바코드 라이브러리
- **KTAS**: 표준화된 분류 프롬프트 + 참조 데이터

## 안전 원칙

- AI 제안은 반드시 의료진 최종 확인 후 실행
- 약물 용량 계산 결과에 신뢰도 수치 표시
- 오류 발생 시 즉각적인 폴백 (수동 프로토콜 참조)
