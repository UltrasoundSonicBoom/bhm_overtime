# 협업 인텔리전스 (Collaborative Intelligence) — 병원 AI 패턴

여러 직군·부서의 정보를 통합하고 협업을 AI가 조율하는 패턴.

## 핵심 시나리오

| 협업 유형 | 주요 기능 | 기대 효과 |
|---------|---------|---------|
| 다학제 케어플랜 | 의사·간호·영양·PT 노트 통합 | 충돌 감지, 일관된 계획 |
| 부서간 의뢰서 | 환자 요약·의뢰사유 자동 생성 | 의뢰서 작성 시간 70% 단축 |
| 환자 교육자료 | 문해력·진단·처방 맞춤 생성 | 환자 이해도 향상 |
| 보호자 커뮤니케이션 | 다국어 통역·의료 용어 쉬운 설명 | 외국인 환자 접근성 |

## 적용 직종

- **의사·전공의** (케어플랜, 의뢰서)
- **간호사** (케어플랜 조율, 환자 교육)
- **행정사무직** (다국어 안내)
- **관리자·파트장** (다학제 팀 조율)

## Claude Code 구현 가이드

```
다학제 케어플랜 통합 패턴:
1. 각 직군 노트 수집 (EMR API 또는 수동 입력)
2. 직군별 관점 요약 (의학적·간호·영양·재활)
3. 상충 항목 감지 및 플래그 처리
4. 통합 케어플랜 초안 생성
5. 팀 리뷰 후 확정
```

## 주요 기술 스택

- **텍스트 통합**: Claude 3.5 (긴 컨텍스트 처리)
- **다국어**: CLOVA Translation API (한국어 특화)
- **충돌 감지**: 구조화된 비교 프롬프트
- **교육자료 생성**: 난이도 파라미터화된 템플릿

## 개인정보 고려사항

- 환자 데이터는 부서간 공유 최소화 원칙
- 다국어 번역 시 외부 API 전송 전 PII 마스킹
- `/smate-b6-masking` 패턴 적용 권장
