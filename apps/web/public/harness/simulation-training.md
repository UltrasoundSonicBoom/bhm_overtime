# 시뮬레이션·훈련 (Simulation & Training) — 병원 AI 패턴

실제 사례 기반 롤플레이·시뮬레이션으로 임상·커뮤니케이션 역량을 훈련하는 AI 패턴.

## 핵심 시나리오

| 훈련 유형 | 대상 | 핵심 기능 |
|---------|------|---------|
| 코드블루 시뮬레이터 | 간호사·전공의 | CPR 프로토콜 순서 피드백, 실시간 오류 지적 |
| 환자 설명 코치 | 전공의·간호사 | 진단 전달 롤플레이, 공감·명료성 점수 |
| 악성 민원 시뮬레이터 | 전 직군 | De-escalation 기술, 감정 조절 피드백 |
| 판독 훈련 어시스턴트 | 전공의·의료기사 | X-ray·CT 힌트 제공, 단계적 추론 유도 |
| 낙상·욕창 예방 훈련 | 간호사 | 케이스 스터디 기반 위험 사정 연습 |

## 적용 직종

- **간호사** (신규 오리엔테이션, 역량 유지)
- **의사·전공의** (환자 커뮤니케이션, 영상 판독)
- **의료기사·보건직** (영상 판독 보조)
- **관리자·파트장** (민원 대응 훈련 기획)

## Claude Code 구현 가이드

```
시나리오 생성 프롬프트 패턴:
1. 환자 프로파일 생성 (나이·진단·복잡도 파라미터)
2. 시나리오 전개 (정상 흐름 + 2가지 오류 분기)
3. 사용자 응답 평가 기준 정의
4. 피드백 생성 (구체적 개선점 + 모범 예시)
```

## 주요 기술 스택

- **롤플레이 엔진**: Claude 3.5 (멀티턴 대화, 역할 유지)
- **의료 영상 분석**: GPT-4o vision / Claude 3.5 vision
- **평가 루브릭**: 구조화된 채점 프롬프트

## 훈련 효과 측정

- 시도 횟수 대비 오류율 추이 기록
- 복잡도별 성공률 (쉬움·보통·어려움)
- 직군별 취약 영역 식별
