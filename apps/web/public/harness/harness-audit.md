# 의무기록 감사 팀 하네스

의무기록의 정확성·코딩 오류·누락 항목을 에이전트팀이 자동으로 감사하고 보고서를 생성한다.

## 에이전트 구성

```
🧠 오케스트레이터
    │
    ├── 📝 기록분석 에이전트
    │     EMR 기록 완결성 검사
    │     필수 항목 누락 감지 (진단·처방·퇴원요약)
    │
    ├── ✅ 코딩검증 에이전트
    │     ICD-10·KCD 코드 정확성 검증
    │     DRG 분류 적절성 확인
    │
    └── 📄 보고서생성 에이전트
          감사 결과 요약 보고서
          개선 필요 항목 우선순위 정렬
```

## 실행 흐름

```
/audit-records [기간] [병동 or 진료과]

Step 1: 대상 기록 추출 (EMR API)
Step 2: 기록분석 에이전트 → 완결성 스코어
Step 3: 코딩검증 에이전트 → 코딩 오류 목록
Step 4: 보고서생성 에이전트 → 감사 보고서
Step 5: 오케스트레이터 → 통합 검토

처리 속도: 100건/4시간 (기존 2일)
```

## 감사 기준

- 퇴원요약서 24시간 내 작성 여부
- 수술기록 즉시 작성 여부
- 동의서·설명서 완비 여부
- 진단코드 주진단·부진단 정확성

## 적용 직종

- **관리자·파트장** (병동 기록 품질 관리)
- **행정사무직** (의무기록팀)
