Head Nurse Workflow

준비 → 생성 → 수정 → 배포

왼쪽은 준비, 가운데는 후보안과 보드, 오른쪽은 수정과 검토입니다. 지금 필요한 일만 골라서 움직일 수 있게 정리했습니다.

  1. 1. 팀과 기간을 고르고 데이터셋 상태를 확인합니다.
  2. 2. AI 초안을 만들고 후보안을 나란히 비교합니다.
  3. 3. 셀 고정, 이벤트 수정, 규정 위반을 정리합니다.
  4. 4. 검증을 통과하면 배포하고 팀 보드에서 확인합니다.
AI와 만드는 영역 왼쪽 제어 영역에서 팀과 기간을 고른 뒤 초안을 생성합니다.
직접 손보는 영역 가운데 보드와 오른쪽 인스펙터에서 셀, 이벤트, 핀을 수정합니다.
배포 전 기준 검증 큐와 규정 위반이 정리된 뒤에만 배포 버튼을 누릅니다.
