# 간호사 근무표 빌더 - 전체 사이트맵

## 📋 전체 구조

```
┌─ 홈 (/)
│
├─ 관리자 (Admin) 역할
│  ├─ 대시보드 (/admin/dashboard)
│  ├─ 근무표 관리 (/admin/schedules)
│  │  ├─ 근무표 목록 (/admin/schedules)
│  │  ├─ 근무표 상세 (/admin/schedules/:id)
│  │  ├─ 근무표 편집 (/admin/schedules/:id/edit)
│  │  └─ 새 근무표 생성 (모달)
│  ├─ 간호사 및 병동 관리 (/admin/profiles)
│  │  ├─ 간호사 목록
│  │  ├─ 병동 관리
│  │  └─ 간호사 상세 정보
│  ├─ 요청 관리 (/admin/requests)
│  │  ├─ 오프 신청 목록
│  │  ├─ 근무 교환 요청 목록
│  │  └─ 요청 상세 (승인/거절)
│  └─ 분석 대시보드 (/admin/analytics)
│     ├─ 공정성 지표
│     ├─ 간호사별 통계
│     └─ 병동별 통계
│
├─ 간호사 (User) 역할
│  ├─ 대시보드 (/nurse/dashboard)
│  ├─ 내 근무표 (/nurse/my-schedule)
│  │  ├─ 월별 근무표 조회
│  │  └─ 근무 상세 정보
│  ├─ 오프 신청 (/nurse/off-request)
│  │  ├─ 오프 신청 폼
│  │  └─ 신청 이력
│  ├─ 근무 교환 (/nurse/shift-swap)
│  │  ├─ 교환 요청 폼
│  │  └─ 요청 이력
│  └─ 알림 (/nurse/notifications)
│     └─ 알림 이력 조회
│
└─ 공통
   ├─ 로그인 (/login)
   ├─ 프로필 (/profile)
   └─ 404 (/404)
```

## 🎯 각 페이지별 상세 구조

### 1. 홈 (/)
**역할:** 랜딩 페이지 / 역할 선택
- 로그인 상태 확인
- 역할별 대시보드로 리다이렉트
- 미로그인 시 로그인 유도

### 2. 관리자 대시보드 (/admin/dashboard)
**컴포넌트:**
- 상단: 통계 카드 (총 간호사, 근무표, 오프/교환 요청)
- 중단: 최근 근무표 목록 (미리보기)
- 하단: 최근 요청 목록 (오프, 교환)
- 우측: 빠른 작업 버튼 (새 근무표, 요청 관리)

### 3. 근무표 목록 (/admin/schedules)
**컴포넌트:**
- 필터: 병동, 연도, 월, 상태
- 테이블: 근무표 목록 (병동, 연월, 상태, 생성일, 액션)
- 액션: 상세 보기, 편집, 삭제, 다운로드

### 4. 근무표 상세 (/admin/schedules/:id)
**컴포넌트:**
- 헤더: 병동명, 연월, 상태
- 간트 차트: 근무표 시각화
- 통계: 간호사별 근무 현황
- 액션: 편집, 확정, 다운로드

### 5. 근무표 편집 (/admin/schedules/:id/edit)
**컴포넌트:**
- 편집 가능한 간트 차트
- 제약 조건 검증 알림
- 근무 교환 다이얼로그
- 변경 사항 추적
- 저장/취소 버튼

### 6. 간호사 및 병동 관리 (/admin/profiles)
**컴포넌트:**
- 탭: 간호사 / 병동
- 간호사 탭:
  - 간호사 목록 (이름, 자격, 경력, 병동)
  - 추가/삭제 버튼
  - 상세 정보 모달
- 병동 탭:
  - 병동 목록 (이름, 인원, 수간호사)
  - 추가/삭제 버튼
  - 병동별 간호사 리스트

### 7. 요청 관리 (/admin/requests)
**컴포넌트:**
- 탭: 오프 신청 / 근무 교환
- 각 탭:
  - 필터: 상태 (대기, 승인, 거절)
  - 테이블: 요청 목록
  - 상세 모달: 승인/거절 버튼

### 8. 분석 대시보드 (/admin/analytics)
**컴포넌트:**
- 공정성 지표:
  - 간호사별 야간 근무 횟수 (차트)
  - 간호사별 휴무일 (차트)
  - 주말 근무 분포 (차트)
- 간호사별 통계 (테이블)
- 병동별 통계 (테이블)

### 9. 간호사 대시보드 (/nurse/dashboard)
**컴포넌트:**
- 상단: 이번 달 통계 (근무일, 야간근무, 휴무)
- 중단: 이번 달 근무표 미리보기
- 하단: 최근 알림 (오프 신청, 교환 요청 결과)
- 빠른 작업: 오프 신청, 교환 요청

### 10. 내 근무표 (/nurse/my-schedule)
**컴포넌트:**
- 월 선택 네비게이션
- 간트 차트 (읽기 전용)
- 근무별 상세 정보 (클릭 시)
- 통계: 이번 달 근무 현황

### 11. 오프 신청 (/nurse/off-request)
**컴포넌트:**
- 오프 신청 폼 (날짜 선택, 사유)
- 신청 이력 테이블
- 상태 표시 (대기, 승인, 거절)

### 12. 근무 교환 (/nurse/shift-swap)
**컴포넌트:**
- 교환 요청 폼 (상대방 선택, 날짜 선택)
- 요청 이력 테이블
- 상태 표시 (대기, 승인, 거절)

### 13. 알림 (/nurse/notifications)
**컴포넌트:**
- 알림 목록 (시간순)
- 필터: 읽음/미읽음
- 알림 상세 정보
