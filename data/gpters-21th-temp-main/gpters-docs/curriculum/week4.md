# Week 4: 최종 발표 + 회고 ("성과 공유 + 다음 단계")

## 목표

- 4주간의 프로젝트 결과를 정리하고 발표하기
- 서로의 경험에서 배울 점 찾기
- 스터디 이후에도 계속 활용할 수 있는 방향 잡기

## 모임 아젠다 (1시간)

1. 최종 발표 — 각자 프로젝트 데모 + 배운 점 (30분, 1인 5분)
2. 베스트 사례 투표 + 축하 (10분)
3. 회고: 잘된 점 / 아쉬운 점 / 다음에 해보고 싶은 것 (15분)
4. 마무리 — 이후 활용 팁 + 커뮤니티 안내 (5분)

## 사용 스킬

- /gpters-thinking-partner — 4주 회고 정리 (잘한 점, 배운 점, 다음 단계)
- /gpters-writer — 최종 사례글 작성
- /gpters-research-case — 다음 단계로 도전할 만한 사례 탐색

## 과제

1. 프로젝트 최종 결과물 정리 (스크린샷 + 핵심 코드 + 느낀 점)
2. `/gpters-writer`로 "4주간의 Claude Code 여정" 최종 사례글 작성 → GPTers 게시
3. (선택) 만든 스킬을 다른 멤버에게 공유하기
