# Week 1: 스타터킷 적용 + 프로젝트 정의 ("나만의 자동화 아이디어 찾기")

## 목표

- Claude Code 스타터킷을 내 프로젝트 폴더에 적용하기
- 4주간 진행할 개인 프로젝트 아이디어 정하기
- Claude Code와 대화하며 아이디어를 구체화하는 경험 쌓기

## 모임 아젠다 (1시간)

1. 자기소개 + 기대하는 것 공유 (10분)
2. 스타터킷 설치 라이브 데모 — `/gpters-setup` 실행 과정 함께 보기 (15분)
3. "내가 자동화하고 싶은 것" 브레인스토밍 — Claude Code에게 물어보며 아이디어 탐색 (20분)
4. 프로젝트 아이디어 발표 + 피드백 (15분)

## 사용 스킬

- /gpters-setup — 스타터킷을 내 프로젝트에 적용 (CLAUDE.md, 폴더 구조 생성)
- /gpters-thinking-partner — 프로젝트 아이디어를 Claude Code와 함께 구체화

## 과제

1. `/gpters-setup`으로 본인 프로젝트에 스타터킷 적용하기 (스크린샷 캡처)
2. `/gpters-thinking-partner`로 프로젝트 아이디어 탐색 대화 나누기 (최소 3번 왕복)
3. `/gpters-writer`로 "Claude Code 스타터킷 첫 적용기" 사례글 작성 → GPTers 게시

## 팁

- 프로젝트는 거창하지 않아도 됩니다. "매일 반복하는 귀찮은 일"에서 시작하세요.
- 예시: 회의록 정리, 이메일 템플릿 생성, 데이터 정리, 블로그 글 초안 등
- Claude Code에게 "나는 ~을 자주 하는데, 자동화할 수 있을까?"라고 물어보세요.
