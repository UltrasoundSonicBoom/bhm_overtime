## \\"ChatGPT도 쓰고, Claude도 쓰고, Gemini도 쓰는데… 따로따로 하고 계신 거예요?\\"

![\"당황](https://tribe-s3-production.imgix.net/4vrZ1m9ZStS6ZV7rKAjFB?auto=compress,format)

  
리서치하려면 Perplexity, 코딩하려면 Claude, 이미지 만들려면 ChatGPT, 데이터 분석하려면 Gemini. AI 도구를 여러 개 쓰다 보면 창만 10개씩 열려 있죠.

Perplexity Computer는 이 문제를 정면으로 해결해요. **19개 AI 모델을 하나의 시스템에서 자동으로 조율** 하는 에이전틱 AI 워크스페이스예요. 목표만 말하면 알아서 작업을 나누고, 최적의 모델을 골라 실행해요. 이 글에서 퍼플렉시티 컴퓨터의 핵심 기능 6가지를 낱낱이 정리해드릴게요.

## Perplexity Computer가 뭔가요?

Perplexity Computer는 2026년 2월에 출시된 **범용 디지털 워커** 예요.  
단순한 챗봇이 아니라, 목표를 주면 알아서 작업을 쪼개고 서브 에이전트를 만들어 실행하는 시스템이에요.

핵심을 정리하면:

- **19개 AI 모델 통합**: GPT, Claude, Gemini 등 최신 모델을 자동으로 라우팅
- **에이전틱 실행**: 목표만 말하면 작업을 분해하고 서브 에이전트가 알아서 처리
- **400+ 앱 연동**: Google, Slack, Notion, GitHub 등과 바로 연결
- **지속 메모리**: 세션이 바뀌어도 이전 작업을 기억하고 이어서 진행

ChatGPT나 Claude가 \\"대화 도구\\"라면, Perplexity Computer는 **\\"일 처리 시스템\\"** 에 가까워요.

## 핵심 기능 6가지

![\"당황이라는](https://tribe-s3-production.imgix.net/s6baZGRiRfp4GZFf3sP9f?auto=compress,format)

### 기능 1: 멀티모델 자동 라우팅 — 최적의 AI가 알아서 붙어요

Perplexity Computer의 가장 강력한 차별점이에요. Claude Opus가 오케스트레이터 역할을 하면서, 들어온 작업의 성격을 분석하고 **19개 모델 중 최적의 모델을 자동으로 선택** 해요.

예를 들어:

- 코딩 작업 → GPT-5.3-Codex 서브 에이전트가 투입
- 이미지 생성 → DALL-E나 Stable Diffusion 모델이 투입
- 리서치 → Perplexity 자체 검색 엔진이 투입
- 문서 작성 → Claude나 GPT가 투입

사용자가 모델을 일일이 고를 필요가 없어요. \\"경쟁사 분석 보고서 만들어줘\\"라고 하면, 검색 모델이 자료를 모으고, 분석 모델이 정리하고, 문서 모델이 보고서를 완성해요.

### 기능 2: 서브 에이전트 시스템 — 작업을 자동으로 쪼개요

하나의 큰 목표를 주면, Computer가 이걸 **태스크와 서브태스크로 자동 분해** 해요. 각 태스크마다 전담 서브 에이전트가 배정되고, 웹 리서치·문서 생성·데이터 처리·API 호출 등을 병렬로 실행해요.

이 워크플로우는 몇 시간에서 **몇 달까지** 지속 실행될 수 있어요. \\"매주 월요일마다 경쟁사 가격 변동을 모니터링해서 보고서 보내줘\\" 같은 장기 작업도 가능하다는 뜻이에요.

### 기능 3: 400+ 앱 커넥터 — 내 도구들과 바로 연결

Google Workspace, Slack, Notion, GitHub, Jira, Salesforce 등 **400개 이상의 앱과 연동** 돼요. 따로 API를 설정하거나 Zapier를 거칠 필요 없이, Computer 안에서 바로 연결해서 쓸 수 있어요.

실전 예시:

- Slack에서 받은 피드백을 자동으로 정리해서 Notion에 기록
- Google Sheets의 데이터를 분석해서 보고서 생성
- GitHub 이슈를 자동으로 생성하고 PR까지 관리

### 기능 4: 지속 메모리 — 내 작업을 기억해요

일반적인 AI 챗봇은 대화가 끝나면 맥락이 날아가요. Perplexity Computer는 다릅니다. **세션 간 작업 내역, 개인 설정, 파일을 모두 유지** 해요.

이게 왜 중요하냐면:

- 어제 작업하던 보고서를 오늘 이어서 수정 가능
- 내가 자주 쓰는 문서 형식을 기억하고 자동 적용
- 프로젝트별 컨텍스트를 유지해서 반복 설명 불필요

수백 개의 활성 프로젝트를 동시에 관리하는 것도 가능해요.

### 기능 5: Skills — 재사용 가능한 워크플로우

최근 업데이트로 추가된 **Skills** 기능이에요. 자주 쓰는 워크플로우를 SKILL.md 파일로 정의해두면, Computer가 상황에 맞게 알아서 꺼내 써요.

예를 들어 \\"PR 리뷰\\" 스킬을 만들어두면, 코드 리뷰 요청이 올 때마다 같은 기준과 절차로 리뷰를 진행해요. Codex Skills이나 Claude Code Skills과 같은 Agent Skills 표준을 따르고 있어요.

### 기능 6: Personal Computer — 24시간 돌아가는 로컬 AI

가장 최근에 추가된 기능이에요. **Mac mini에서 24시간 연속 실행되는 개인 AI 환경** 이에요.

특징:

- 로컬 파일, 앱, 세션에 직접 접근
- 항상 켜져 있어서 언제든 작업 가능
- Perplexity의 보안 인프라에서 실행되므로 데이터가 안전

클라우드 기반 Computer가 \\"요청하면 일하는 AI\\"라면, Personal Computer는 **\\"항상 대기하면서 알아서 돌아가는 AI\\"** 예요.

Perplexity Computer의 진짜 가치는 \\"19개 모델\\"이라는 숫자에 있지 않아요. **모델 선택을 사용자가 하지 않아도 된다** 는 점이 핵심이에요.

지금까지 AI 도구를 쓸 때 가장 큰 비용은 \\"어떤 도구를 언제 쓸지 판단하는 시간\\"이었어요. 코딩은 Claude가 낫고, 검색은 Perplexity가 낫고, 이미지는 DALL-E가 낫다는 걸 알아도, 매번 전환하고 컨텍스트를 옮기는 게 실제 작업 시간보다 더 걸릴 때가 많았죠.

Computer는 이 판단 비용을 제로로 만들어요. \\"경쟁사 분석해서 PPT 만들어줘\\"라고 하면, 검색 모델이 자료를 모으고, 분석 모델이 정리하고, 문서 모델이 PPT를 만들어요. 사용자는 결과만 확인하면 돼요.

다만 월 $89~$200이라는 가격은 개인 사용자에게 부담이에요. 이 비용이 정당화되려면 **주 3회 이상 복합 워크플로우를 실행하는 사람** — 마케터, 리서처, PM — 이어야 해요. 단순 검색이나 챗봇 용도라면 무료 Perplexity Pro면 충분해요.

AI 에이전트 시대의 핵심 질문은 \\"어떤 모델이 좋은가\\"가 아니라 **\\"어떤 시스템이 내 일을 대신해주는가\\"** 예요. Perplexity Computer는 그 질문에 가장 직접적으로 답하는 도구 중 하나예요.
