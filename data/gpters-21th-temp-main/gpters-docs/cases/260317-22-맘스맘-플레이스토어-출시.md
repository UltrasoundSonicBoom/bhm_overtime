**2.정식 출시하기**

이제 커서를 켜서 기존에 쓰던 chat에 이제 뭐하면 되는지 물어봅니다.

```
완료됨. 이제 뭐하면 됨??\n\nProduction access 승인된 거 축하해요. 이제 정식 출시만 진행하면 됩니다.\n1. Production 릴리즈 만들기\nPlay Console → 앱 선택 → Test and release → Production\nCreate new release (또는 새 릴리즈 만들기)\nApp bundle 업로드\n경로: momsmom/build/app/outputs/bundle/release/app-release.aab\n이미 같은 versionCode로 다른 트랙에 올렸다면, pubspec.yaml에서 version: 1.0.0+N의 N을 1 올리고 다시 빌드한 뒤 업로드\nRelease notes 작성 (예: \"첫 정식 출시\")\nStaged rollout 설정\n처음엔 5% 또는 10%로 시작\n크래시/ANR/리뷰 확인 후 25% → 50% → 100%로 확대\n2. 출시 전 확인\nStore listing: 앱 이름, 설명, 스크린샷, 피처 그래픽, 아이콘\nApp content: Privacy policy URL, Data safety, Content rating, Ads, Target audience\n위 항목이 비어 있으면 Review release 단계에서 막힐 수 있음\n3. 출시 실행\nReview release → Start rollout to Production (또는 프로덕션에 롤아웃 시작)\n승인되면 설정한 비율(예: 5%)에 맞춰 사용자에게 노출됨\n4. 출시 후\nAndroid vitals(Crash/ANR)와 리뷰를 며칠간 자주 확인\n문제 없으면 rollout 비율을 점진적으로 올리기\n여기까지 진행하다가 막히는 화면이나 에러가 있으면 그 부분만 알려주면 됩니다.
```
