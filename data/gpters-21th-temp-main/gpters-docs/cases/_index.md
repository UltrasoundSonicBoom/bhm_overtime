## 파일 네이밍 규칙

```
[기수]-[번호]-[제목].md
```

- **기수**: 스터디 기수 (예: `20th`, `21th`, `ax`)
- **번호**: 해당 기수 내 순번 (1, 2, 3...)
- **제목**: kebab-case 제목

### 예시

```
20th-1-slack-error-auto-classify.md
21th-1-pr-review-summary.md
```
