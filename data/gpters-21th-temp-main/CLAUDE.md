## 폴더 구조

- `gpters-docs/` — 제공 콘텐츠 (수정 X)
  - `gpters-docs/cases/` — GPTers 사례 DB
  - `gpters-docs/curriculum/` — 주차별 커리큘럼
  - `gpters-docs/templates/` — 양식 (정의서, 과제, 스킬)
  - `gpters-docs/tips/` — Claude Code 팁
- `my/` — 내 공간 (자유)
  - `my/projects/` — 프로젝트 정의서
  - `my/cases/` — 내가 쓴 사례글
  - `my/research/` — 팁 검색 결과
  - `my/notes/` — 자유 메모
