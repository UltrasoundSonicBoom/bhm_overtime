# GPTers Claude Code 스터디 스타터킷

## 빠른 시작

### 1. 압축 해제

이 폴더를 자기 프로젝트 루트에 복사하세요.

### 2. 온보딩

터미널에서 `claude`를 실행하고 `/gpters-setup` 을 입력하세요.

### 3. 스킬 사용

| 스킬 | 용도 | 언제 쓰나 |
|------|------|----------|
| `/gpters-21th-hello` | 첫 체험 + 전자책 생성 | **1주차 라이브에서 가장 먼저** |
| `/gpters-setup` | 온보딩 + 환경 세팅 | 처음 시작할 때 |
| `/gpters-thinking-partner` | 요구사항 정리 | 뭘 만들지 정하고 싶을 때 |
| `/gpters-research-case` | 사례 탐색 | 다른 사람들 사례 구경할 때 |
| `/gpters-writer` | 사례글 작성 | 매주 과제 제출할 때 |
| `/deep-research` | 팁 검색 | 막혔을 때 |
| `/skill-creator` | 스킬 만들기 | 나만의 자동화를 만들고 싶을 때 |
| `/gpters-skill-updater` | 스킬 수정 | 만든 스킬 고치고 싶을 때 |

## 폴더 구조

```
gpters-docs/          ← 제공 콘텐츠 (수정 X)
  ├── cases/          ←   GPTers 사례 DB
  ├── curriculum/     ←   주차별 안내
  ├── templates/      ←   양식
  └── tips/           ←   Claude Code 팁

my/                   ← 내 공간 (자유)
  ├── projects/       ←   프로젝트 정의서
  ├── cases/          ←   내가 쓴 사례글
  ├── research/       ←   팁 검색 결과 축적
  └── notes/          ←   자유 메모
```
