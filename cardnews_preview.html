UX 최종 구조

덱 목록 → 선택 → 세로 스크롤 카드 탐색. 고스트 바로 인접 덱 전환. X로 목록 복귀.

01. 3가지 핵심 변경

1
초기화면: 덱 목록 + AI 3줄 요약
카드슬라이더가 바로 나오지 않는다. 덱 목록이 먼저. 각 카드에 AI 생성 3줄 요약 불릿. "볼 가치 있나" 판단 후 선택.
2
선택 → 포커스뷰 (전체화면)
선택 덱이 전체 영역으로 전환. 나머지 덱은 위아래 고스트 바 (반투명, 제목만). 고스트 탭 → 해당 덱으로 이동.
3
세로 스크롤 = 카드 이동 (snap)
포커스뷰에서 위아래 스크롤로 카드 1→5 이동. scroll-snap-stop: always로 카드 단위 정확히 멈춤. 상단 X 버튼 → 목록 복귀.

02. 실제 적용 시 체크리스트

cardnews.js 구현 포인트
openFocus(id) — view 전환 + ghost 업데이트 + 카드 높이 설정
closeFocus() — 포커스뷰 → 목록뷰 복귀
ghostNav(dir) — 인접 덱 전환 (prev/next)
scroll → Math.round(scrollTop/cardH)로 도트 업데이트
카드 높이: focusCards.clientHeight를 .fc 각각에 style.height 설정
cardnews-pipeline.js: prompt에 summary[] 3줄 반환 지시 추가
