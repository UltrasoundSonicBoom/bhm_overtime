C. 전용 크롬 탭 (독립형)

팝업 대신 전용 Chrome 탭(extension page)으로 동작. snuhmate.com 없이 chrome.storage에 직접 저장 → Drive 동기화.

장점

  • 공간 제한 없음
  • snuhmate.com 불필요
  • 캘린더·히스토리 풍부하게 표시

단점

  • 개발량 가장 많음
  • 데이터 모델 재구현 필요

⚡ 동기화 지연시간 (모든 옵션 공통)

구간 지연 설명
확장 → 로컬 저장 <50ms chrome.storage.local 즉시 저장
로컬 → Drive 동기화 3초 (디바운스) SyncManager 3초 디바운스 후 Drive API 호출
Drive → 다른 기기 반영 탭 포커스 시 pullOnResume() — 탭 복귀 20초 쿨다운
최악의 경우 ~25초 3초 디바운스 + Drive API 지연 + 다음 탭 복귀까지
