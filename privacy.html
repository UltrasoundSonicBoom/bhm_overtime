제1조 총칙

SNUH 메이트(이하 "서비스" 또는 "앱")는 서울대학교병원, 보라매병원 등 의료기관 종사자가 연차 휴가, 시간외 근무, 급여명세서 등의 근무 기록을 스스로 관리할 수 있도록 지원하는 웹 애플리케이션입니다.

운영자(이하 "개발자")는 「개인정보 보호법」, 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」을 준수하며, 이용자의 개인정보를 보호하기 위해 본 방침을 수립·공개합니다.

제3조 개인정보의 처리 목적

  1. 사용자 본인의 근무 기록(연차·시간외·급여) 계산 및 관리 기능 제공
  2. Google 계정 연동을 통한 기기 간 데이터 백업 및 복원 (선택 기능)
  3. Google Calendar 연동을 통한 휴가 일정 자동 등록 (선택 기능)
  4. 서비스 오류 개선 및 신규 기능 개발을 위한 익명 통계 분석

제4조 개인정보의 보유·이용 기간 및 파기

로컬 데이터

브라우저 로컬 스토리지에 저장된 데이터는 사용자가 직접 삭제하거나 브라우저 데이터를 초기화할 때 삭제됩니다. 해당 데이터는 사용자 기기 내 브라우저 저장소에만 존재하며, 개발자는 이를 자체 서버에 별도로 보관하지 않습니다.

Google Drive 백업 데이터

사용자 본인의 Google Drive appDataFolder에 저장된 데이터는 사용자가 Google Drive에서 직접 삭제하거나, 앱의 "연결 해제" 기능을 사용하면 앱 데이터가 삭제됩니다. Google의 데이터 보관 정책이 적용됩니다.

Google Calendar 이벤트

앱이 생성한 캘린더 이벤트는 사용자가 Google Calendar에서 직접 삭제하거나 앱의 휴가 기록 삭제 시 함께 삭제됩니다.

제5조 개인정보의 제3자 제공

🚫 개발자는 사용자의 근무 기록 원문을 광고, 판매 또는 제3자 제공 목적으로 이용하지 않습니다.

다만, 사용자가 선택적으로 Google 연동을 활성화한 경우, 해당 데이터는 사용자 본인의 Google 계정으로 직접 전송되며, Google의 개인정보처리방침(policies.google.com/privacy)이 적용됩니다.

제7조 정보주체의 권리·의무 및 행사 방법

사용자는 언제든지 다음 권리를 행사할 수 있습니다.

  • 열람권: 앱 내 데이터는 사용자 기기에서 직접 확인 가능합니다.
  • 정정·삭제권: 앱 내 기능 또는 브라우저 데이터 삭제를 통해 직접 처리 가능합니다.
  • 처리 정지권: Google 연동 해제 시 추가 데이터 전송이 중단됩니다.
  • Google 연동 해제: 앱의 "연결 해제" 버튼 또는 Google 계정 → 보안 → 타사 앱 접근에서 언제든 철회 가능합니다.

제8조 Google API 서비스 이용 공개

SNUH 메이트의 Google API 서비스 이용 및 데이터 공개는 Google API 서비스 사용자 데이터 정책을 포함한 Google의 제한적 사용 정책을 준수합니다.

  • Google API를 통해 수집된 데이터는 앱의 기능 제공(백업·복원·일정 등록) 목적에만 사용됩니다.
  • Google API를 통해 수집된 데이터는 광고, AI 모델 학습, 제3자 판매 등에 사용되지 않습니다.
  • Google Drive에 저장되는 데이터는 사용자 본인의 계정 내 앱 전용 폴더(appDataFolder)에만 저장됩니다.
  • 앱이 수집한 Google 사용자 데이터는 사용자의 명시적 동의 없이 다른 목적으로 사용하거나 공유하지 않습니다.

제9조 쿠키 및 유사 기술

본 서비스는 쿠키(Cookie) 대신 브라우저 로컬 스토리지(localStorage)를 사용합니다. Google Analytics는 분석 목적으로 쿠키를 사용할 수 있으며, 브라우저 설정에서 비활성화 가능합니다.

제10조 개인정보의 안전성 확보 조치

  • 서버 미저장: 사용자 근무 기록은 개발자 서버에 저장되지 않습니다.
  • HTTPS 암호화: 모든 통신은 TLS/HTTPS로 암호화됩니다.
  • OAuth 2.0 표준: Google 인증은 업계 표준 OAuth 2.0을 사용하며, 비밀번호를 수집하거나 저장하지 않습니다.
  • 최소 권한: Google 권한은 사용자가 기능을 활성화할 때만 개별적으로 요청합니다(Incremental Authorization).
  • Access Token 메모리 보관: Google Access Token은 서버에 저장하지 않고 브라우저 메모리에만 임시 보관합니다.

제11조 AI·자동화 도구의 데이터 수집 금지

본 서비스의 콘텐츠, 사용자 데이터, 서비스 구조는 다음 행위를 명시적으로 금지합니다.

  • AI 언어 모델(LLM) 학습을 위한 데이터 수집(스크래핑, 크롤링)
  • 자동화 도구를 이용한 콘텐츠 복제 또는 모방
  • 사용자 입력 데이터의 무단 수집 및 제3자 제공
  • 서비스의 기능·디자인·알고리즘을 역공학(reverse engineering)하여 유사 서비스 제작

본 서비스는 robots.txt 및 meta 태그를 통해 AI 크롤러의 접근을 기술적으로 차단하고 있습니다. 이를 우회하는 행위는 「저작권법」 및 「정보통신망법」 위반에 해당할 수 있습니다.

제12조 만 14세 미만 아동

본 서비스는 의료기관 종사자를 대상으로 하며, 만 14세 미만 아동의 서비스 이용을 허용하지 않습니다. 만 14세 미만 아동의 개인정보가 수집된 사실을 발견하면 즉시 삭제 조치합니다.
