# Spec: B1 -- content/ops Directory Conventions

## Goal

`content/`와 `ops/` 하위 디렉토리의 목적을 문서화하여 운영자가 원본 저장 위치를 찾을 수 있게 한다.

## Acceptance Criteria

1. `content/README.md` 존재 및 디렉토리 목적 문서화
2. `ops/README.md` 존재 및 디렉토리 목적 문서화
3. MD 원본 vs 운영 프롬프트 구분 명확
4. 운영자가 원본 문서 저장 위치를 찾을 수 있어야 함

## Files
- content/README.md (이미 존재, 검증)
- ops/README.md (이미 존재, 검증)
