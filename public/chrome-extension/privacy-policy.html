SNUH Mate Companion 개인정보처리방침

SNUH Mate Companion(이하 "확장프로그램")은 사용자의 개인정보를 소중히 여기며, 최소한의 데이터만 처리합니다.

1. 수집하는 정보

항목수집 여부설명
개인 식별 정보 (이름, 이메일 등)수집하지 않음확장프로그램은 사용자의 개인정보를 직접 수집하지 않습니다.
브라우징 히스토리수집하지 않음방문한 웹사이트 기록을 추적하지 않습니다.
최근 다운로드 PDF 정보로컬 저장만PDF 다운로드 완료 시 파일명과 URL을 chrome.storage.local에 임시 저장합니다. 24시간 후 자동 만료되며, 외부로 전송되지 않습니다.
확장프로그램 설정로컬 저장만사용자가 설정한 도메인, 디버그 모드 값을 chrome.storage.local에 저장합니다.

2. 데이터 사용 목적

  • 빠른 기록 (Quick Capture): 사용자가 팝업에서 입력한 초과근무/온콜 정보를 SNUH Mate 웹사이트로 전달하여 기록합니다.
  • PDF 가져오기: 사용자가 선택한 PDF URL의 내용을 가져와 SNUH Mate 급여명세서 파서로 전달합니다.

3. 데이터 공유

확장프로그램은 수집한 데이터를 제3자에게 판매, 공유, 전송하지 않습니다.

모든 데이터는 사용자의 브라우저 내에서만 처리되며, SNUH Mate 웹사이트(snuhmate.com)와의 통신만 수행합니다.

4. 데이터 보안

  • 확장프로그램은 SNUH Mate에 Google 계정으로 로그인된 상태에서만 동작합니다.
  • 모든 통신은 HTTPS를 통해 암호화됩니다.
  • 비즈니스 로직(급여 계산, 초과근무 산정)은 확장프로그램 내에서 실행되지 않으며, SNUH Mate 페이지에 위임됩니다.

6. 사용자 권리

  • 확장프로그램을 언제든 제거할 수 있으며, 제거 시 chrome.storage.local에 저장된 모든 데이터가 자동으로 삭제됩니다.
  • 옵션 페이지에서 설정을 변경하거나 초기화할 수 있습니다.

7. 변경 사항

본 개인정보처리방침이 변경될 경우, 확장프로그램 업데이트를 통해 고지합니다.
